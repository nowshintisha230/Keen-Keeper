# 📌 Project Name
KeenKeeper 
---

## 📖 Short Description
KeenKeeper is a modern friendship tracking web application built with Next.js.  
It helps users manage friends, track interactions (Call, Text, Video), and visualize friendship activity in a clean and responsive UI.

---

## ⚙️ Technologies Used
- Next.js
- App Router
- Recharts
- Tailwind CSS
- Font Awesome Icons
- LocalStorage
- DaisyUI
- Toast

---

## ✨ Key Features
1. 👥 Friend Management System with detailed friend profiles  
2. ⚡ Interaction Timeline for Call, Text, and Video tracking  
3. 📊 Responsive UI with clean design across all devices
